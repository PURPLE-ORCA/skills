# Next.js PayZone Server Action

## CRITICAL: The Localhost Webhook Trap
When developing locally, Next.js runs on `localhost:3000`. PayZone's servers cannot reach your local machine to deliver the webhook confirmation. 
* User-facing redirects (`successUrl`, `failureUrl`, `cancelUrl`) **MUST** use the Next.js app URL (e.g., `NEXT_PUBLIC_APP_URL`).
* The background server-to-server webhook (`callbackUrl`) **MUST** point to the public Convex HTTP edge router (e.g., `CONVEX_HTTP_URL` ending in `.site`), even in development.

## Cryptography Runtime
Unlike the Convex Edge webhook, this is a Next.js `"use server"` action running in a standard Node.js environment. You must use Node's native `crypto` module to generate the SHA-256 signature.

## Payload Structure & Signature
PayZone expects a highly specific JSON payload. The signature is generated by creating a SHA-256 hash of the `PAYZONE_SECRET_KEY` concatenated directly with the stringified JSON payload.

## Implementation Template

```typescript
"use server";

import { createHash } from "crypto";

interface PayZoneInitResponse {
  payload: string;
  signature: string;
  payzoneUrl: string;
}

export async function initiatePayzonePayment(
  bookingReference: string,
  amount: number,
  description: string
): Promise<PayZoneInitResponse> {
  const merchantAccount = process.env.PAYZONE_MERCHANT_ACCOUNT;
  const secretKey = process.env.PAYZONE_SECRET_KEY;
  const payzoneUrl = process.env.PAYZONE_URL;
  const appUrl = process.env.NEXT_PUBLIC_APP_URL;
  const convexHttpUrl = process.env.CONVEX_HTTP_URL;

  if (!(merchantAccount && secretKey && payzoneUrl && appUrl && convexHttpUrl)) {
    throw new Error("Missing required PayZone or Application environment variables");
  }

  const timestamp = Math.floor(Date.now() / 1000);
  const chargeId = timestamp.toString();

  const payload = {
    merchantAccount,
    timestamp,
    skin: "vps-1-vue",
    customerId: crypto.randomUUID(),
    customerCountry: "MA",
    customerLocale: "fr_FR",
    chargeId,
    orderId: bookingReference,
    price: amount.toString(),
    currency: "MAD",
    description,
    mode: "DEEP_LINK",
    paymentMethod: "CREDIT_CARD",
    showPaymentProfiles: "false",
    // THE TRAP: Callback goes to Convex, the rest go to Next.js
    callbackUrl: `${convexHttpUrl}/payzone/webhook`,
    successUrl: `${appUrl}/booking/success?ref=${bookingReference}`,
    failureUrl: `${appUrl}/booking/failure?ref=${bookingReference}`,
    cancelUrl: `${appUrl}/booking/cancel?ref=${bookingReference}`,
  };

  const jsonPayloadString = JSON.stringify(payload);

  const signature = createHash("sha256")
    .update(secretKey + jsonPayloadString)
    .digest("hex");

  return {
    payload: jsonPayloadString,
    signature,
    payzoneUrl,
  };
}
