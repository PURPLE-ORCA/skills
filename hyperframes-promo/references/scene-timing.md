# Scene Timing Reference

Source: Claude Design + HyperFrames guide, verified against real renders.

## Duration by Content Density

| Content Type | Duration | Notes |
|-------------|----------|-------|
| Visual only (icon, hero image, logo) | 1.5–2s | Let the visual breathe |
| 1–3 words (brand name, tagline) | 2–3s | "TIC·TAX", "Know your runway" |
| 4–10 words (short headline) | 3–4s | "Freelance income is a lie." |
| 11–20 words (headline + subtitle) | 4–6s | Split into two visual beats |
| 21–35 words (dense copy) | 6–8s | Rare in promos; split across scenes |
| **Hard ceiling (any text)** | **5s** | Only break with explicit mid-scene motion |

## 30s Promo — Default Scene Allocation

| Scene | Duration | Why |
|-------|----------|-----|
| Hook | 3–4s | Fast — capture attention immediately |
| Problem | 4s | Provocative, one concept |
| Solution | 4s | Key number + breakdown |
| Urgency | 4–5s | Ticking clock, active animation |
| Features | 4–5s | Grid staggered, audience scans |
| Differentiator | 3–4s | Fast punches, 3 bullets |
| CTA | 4–5s | Brand close, slow breath out |

**Total:** ~28–31s. Adjust last scene to hit exactly 30s.

## 15s Promo — Condensed (4 Scenes)

| Scene | Duration |
|-------|----------|
| Hook | 2.5s |
| Problem → Solution | 4s |
| Features | 4s |
| CTA | 4.5s |

## Timing Anti-Patterns

- **First scene too long** → viewers bounce. Hook must be ≤ 4s.
- **Same-duration every scene** → feels robotic. Vary 2–5s.
- **Last scene rushed** → CTA needs room to land. Min 4s.
- **No transition between same-color scenes** → hard cut can feel like a glitch. Add Flash-through-white or a brief 2-frame dissolve.
- **Scenes longer than 5s with no motion** → viewer thinks video froze. Always add Ken Burns or counter for scenes > 4s.
